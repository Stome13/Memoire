🔗 Endpoints API Backend

Ces endpoints sont appelés depuis les pages PHP:

  • POST /backend/api/auth.php?action=login - Connexion
  • POST /backend/api/auth.php?action=logout - Déconnexion
  • POST /backend/api/auth.php?action=register - Inscription
  • GET /backend/api/users.php?action=stats - Stats (Admin)
  • GET /backend/api/users.php?action=profile - Profil utilisateur
  • GET /backend/api/pharmacies.php?action=list - Pharmacies
  • GET /backend/api/reservations.php?action=list - Réservations
🐛 Debugging
  • Ouvrir DevTools (F12) → Console pour JS errors
  • Network tab pour voir les appels API
  • Vérifier logs: backend/logs/
  • Session active? Vérifier: var_dump($_SESSION); dans session.php
